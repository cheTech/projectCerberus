# project "Cerberus" (beta)
- placed at the entrance to the organization
- protects users from strangers
- only need a webcam
# install
```bash
git clone https://github.com/cheTech/projectCerberus.git
cd projectCerberus
python3 -m pip install -r requirements.txt
python3 main.py
```
–êfter that the program will create a settings.json file and it will need to be edited.
# using
```bash
python3 main.py
```
